# Acronym Dictionary

## Umbra-specific
- **V0-C** — “Version 0 - Control plane” (Agent Identity Control Plane MVP slice)
- **Receipt** — audit event emitted for a decision and/or tool invocation (hash-chained; signing-ready)

## Security & architecture
- **PDP** — Policy Decision Point (evaluates allow/deny + obligations)
- **PEP** — Policy Enforcement Point (enforces the PDP decision at the boundary)
- **ABAC** — Attribute-Based Access Control
- **RBAC** — Role-Based Access Control
- **OIDC** — OpenID Connect (SSO/authn protocol)
- **IdP** — Identity Provider (Keycloak, Okta, Azure AD, etc.)
- **mTLS** — mutual TLS (service-to-service authentication)
- **RLS** — Row-Level Security (Postgres feature to enforce tenant isolation at DB level)
- **OTel** — OpenTelemetry (traces/metrics/logs correlation)
- **SLSA** — Supply-chain Levels for Software Artifacts (build provenance guidance)
- **SSDF** — Secure Software Development Framework (NIST)

## Agent/tooling
- **MCP** — Model Context Protocol (tool calling / tool integration interface)
- **CLI** — Command Line Interface (wrapper-based enforcement surface)

## Ops
- **SIEM** — Security Information and Event Management (Splunk, Elastic, etc.)
- **ACV** — Annual Contract Value
